# What's this?

- 넷플릭스 클론 웹사이트
- MERN Stack 이용하여 영화 데이터를 Creat, Read, Update 그리고 Delete할 수 있도록 제작
  - MongoDB
  - ExpressJS
  - ReactJS
  - NodeJS

# Dependencies

## Frontend

- ReactJS
- styled-components
  - `npm install --save styled-components`
  - styled-components reset
    - `npm install styled-reset`
- material-ui
  - `npm i @material-ui/icons @material-ui/core`
